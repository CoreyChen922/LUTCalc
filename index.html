LUTCalc

A Web App for generating, analysing and previewing 1D and 3D Lookup Tables (LUTs) for video cameras that shoot log gammas.

Install LUTCalc

LUTCalc is a Javascript web app designed for offline use. Installation is as follows for different preferred browsers:

Firefox - LUTCalc can be installed from Firefox as a self-contained packaged app which looks like a conventional application by clicking .

On Windows Firefox will place a shortcut to the LUTCalc app on the desktop. Double clicking will open and run it as any other application. On a Mac LUTCalc is installed in the Applications folder, though when running for the first time you will need to ctrl-click and choose 'Open'. The brings up a warning that it is a downloaded app to make sure that you are happy to run it. After the first time it will open like any other application.

Safari - Safari allows very limited file access to Javascript in the browser, so LUTCalc's sister project LUTCalc For Mac creates a native app wrapped around LUTCalc and Safari's webkit engine providing full read and write access and the look of a conventional application.

Chrome - LUTCalc should be installed as a Chrome Web App. Download and open / unpack the zip or tar.gzip files above and then in Chrome either click Windows->Extensions from the menu bar or enter chrome://extensions in the address bar.

On the top right hand side should be a checkbox marked 'Developer Mode'. Make sure that this is ticked. Towards the left should be a button marked 'Load unpacked extension...'. Click that, look for the folder containing the unpacked LUTCalc files and click 'Select'.

You should now see LUTCalc in the list of installed extensions. You can now untick developer mode. LUTcalc can be started by clicking the 'Launch' button here, or from the Chrome App Launcher. The only restriction in Chrome is that files will save to Chrome's download folder.

Internet Explorer - not currently available.

Cinematic Cameras & Log Recording.

Modern video cameras such as Sony's F55, F5 and FS7, Canons CineEOSes and Arri Alexas and Amiras are able to resolve huge dynamic ranges and wide colour spaces, and coupled with the availability of sophisticated grading tools log recording has developed to offer the greatest flexibility of data storage within the limitations of 10, 12 or in the case of the C300 8-bit codecs.

Log recordings attempt to distribute picture information evenly over each recorded stop so that highlights can be manipulated to the same degree as midtones and - allowing for noise - shadows. The downsides are that this results in a flat, bland picture (particularly when combined with a wide colour gamut) and that if using a waveform for exposure, the low contrast log signal is not well suited to easy, accurate exposure judgement.

An elegant solution is that many cameras can use lookup tables or LUTs to map the base log data into more attractive forms designed around displays and their colour space and contrast characteristics. These are typically either base looks provided by manufacturers or LUTs generated from graded material to ensure a consistent look across a production.

In addition, some cameras can 'burn-in' LUTs, recording a 'graded' picture where postproduction resources or time do not lend themselves to a fully log workflow.

LUTCalc The Information Tool

LUTCalc has been written to utilise publically available information about various manufacturers log equations and finishing looks. From that LUTCalc can produce tables and charts of exposure values suitable for a given recording and desired end look and quickly and easily preview how different LUTs will alter the picture using test image.

In addition to the built-in colour spaces and transfer functions, LUTCalc can also read LUTs in the widespread .cube format using the LUTAnalyst tool. Then, if given the expected input gamma and colour space of that LUT it will break down the contrast and colour elements of the LUT and allow them to be used and visualised in the same way as any of the built-in options.

LUTCalc The Adjustment Tool

As well as displaying the basic appearance of different looks, LUTCalc can make a number of adjustments. Output levels can be scaled to preferred values for black and arbitrary reflectances, different colour spaces can be applied to shadows and highlights and the colour temperature can be adjusted, either as a CTO / CTB shift or by directly setting recorded and output temperatures.

Once the input, output and adjustment options have been set LUTCalc can then output .cube LUTs suitable for use in camera as monitor LUTs or in post to provide one-click base looks.
